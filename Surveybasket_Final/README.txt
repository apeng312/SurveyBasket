Project Name: Survey Basket
Group members: Andy Peng(apeng), Kechun Mao(kmao)
Description: Survey basket is a survey website for posting, taking and distributing surveys. It has the credit system to encourage people to take surveys.  
How to operate it: run ./mongod in mongodb folder and run node server.js. (We will possibly host it on heroku).
Modules and Libraries we used: 
backend: mangodb, mangoose, express, nodejs, password
frontend: jQuery, underscore, bootstrap
Link to the website: surveybasket.herokuapp.com
